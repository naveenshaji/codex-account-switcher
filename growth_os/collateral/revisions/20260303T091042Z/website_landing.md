# Codex Account Switcher\n\nSwitch Codex accounts from your macOS menu bar, without manual auth file juggling.\n\n## Built for heavy Codex users\nIf you run long desktop + CLI sessions across multiple accounts, this tool keeps account switching fast and predictable.\n\n## Core workflow\n1. Save multiple accounts\n2. Pick active account in one click\n3. Restart Codex / open new CLI session from the same panel\n4. Monitor 5-hour + weekly remaining usage per account\n\n## Security model\n- Local profile storage with restrictive permissions\n- Atomic writes to `~/.codex/auth.json` on switch\n- Open source implementation for full auditability\n\n## Current limits\nRunning sessions do not hot-reload auth; restart/new session is required.\n\n## OSS repo\n[View source on GitHub](https://github.com/naveenshaji/codex-account-switcher)
