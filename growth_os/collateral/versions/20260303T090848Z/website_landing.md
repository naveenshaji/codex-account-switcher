# Codex Account Switcher\n\nA macOS menu bar app for people who run Codex heavily across multiple accounts.\n\n## Why it exists\nSwitching accounts by editing `~/.codex/auth.json` manually is fragile and slow. This app keeps accounts ready and lets you switch active auth in one click.\n\n## What you get\n- One-click active account switch\n- Account add/remove from menu bar\n- 5-hour + weekly remaining usage bars per account\n- Quick actions to restart Codex desktop and open a new CLI session\n\n## Built for real Codex workflows\nThe app is focused on operator speed, clear state, and predictable switching behavior.\n\n## Open source\n- Repo: https://github.com/naveenshaji/codex-account-switcher\n- Stack: SwiftUI + Swift Package Manager\n\n## Notes\n- Existing sessions do not hot-reload auth; restart/new session is required after switch.\n- Credentials are stored locally with restrictive permissions.\n\n## CTA\n[View source on GitHub](https://github.com/naveenshaji/codex-account-switcher)
