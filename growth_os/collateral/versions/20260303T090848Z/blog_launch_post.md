# Launch: Codex Account Switcher (Open Source)\n\nI built **Codex Account Switcher**, a macOS menu bar utility for people running Codex across multiple accounts.\n\n## The problem\nIf you use Codex desktop + CLI heavily, account switching usually means manually editing `~/.codex/auth.json`, restarting tools, and keeping usage windows in your head.\n\n## What this tool changes\n- Keep multiple accounts saved\n- Switch active account with one click\n- See remaining quota for 5-hour and weekly windows per account\n- Trigger Codex app restart and new CLI session from the same menu\n\n## Why open source\nAuth/account tooling should be inspectable. The repository is public so you can audit behavior and adapt it to your own workflow.\n\n## Current limitations\n- Running sessions do not hot-reload auth\n- Assumes file-backed auth (`~/.codex/auth.json`)\n\n## Try it\n```bash\ngit clone https://github.com/naveenshaji/codex-account-switcher.git\ncd codex-account-switcher\nswift run CodexAccountSwitcherApp\n```\n\nIf you manage multiple Codex accounts daily, this should remove a lot of friction.\n\nRepo: https://github.com/naveenshaji/codex-account-switcher
